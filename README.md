# OrderBot AI

AI-powered WhatsApp order bot for small businesses.

## What it does
OrderBot AI automates the entire order-taking process on WhatsApp. A customer texts the number, the AI reads the order, confirms the items, calculates the total, and sends a Stripe payment link — automatically.

## The problem it solves
Small businesses waste hours every day manually replying to WhatsApp orders, calculating totals, and chasing payments. OrderBot AI replaces that entire workflow.

## Tech stack
- WhatsApp Web.js
- OpenAI GPT-3.5
- Stripe Payments
- Node.js

## How to run
1. Clone the repo
2. Run npm install
3. Add your OpenAI API key to index.js
4. Run node index.js
5. Scan the QR code with WhatsApp
